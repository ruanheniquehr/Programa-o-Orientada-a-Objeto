Sistema de Login

Projeto desenvolvido em Java utilizando Programação Orientada a Objetos (POO).

O sistema possui:

- Login de usuário
- Cadastro de clientes
- Listagem de clientes
- Envio de notificações

## Como Executar o Projeto

### 1. Extrair o arquivo ZIP

Extraia o arquivo do projeto para uma pasta.

Exemplo:

```plaintext
SistemaNotificacao.zip
↓
SistemaNotificacao/
```

---

### 2. Abrir a pasta

Abra a pasta extraída no:

- VS Code
ou
- Prompt de Comando / Terminal

---

### 3. Entrar na pasta do projeto

Exemplo:

```bash
cd SistemaNotificacao
```

---

### 4. Compilar o projeto

Execute:

```bash
javac *.java
```

Esse comando gera os arquivos `.class`.

---

### 5. Executar o sistema

Execute:

```bash
java Main
```
